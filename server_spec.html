“Xeon Server“ (コードネーム:4号機鯖)
稼働開始日: 2021/03/19~
CPU: Intel Xeon E5-2665 8C16T@2.40GHz x2
GPU: Radeon HD 7570
RAM: DDR3 32GB 4Rx4 10600R(1333MHz) x4
Storage: Intel SSD 128GB / WD Green 1.5TB / WD Black 500GB
MB: Jingsha X79 Dual CPU Motherboard (LGA2011)
OS: Ubuntu Linux 20.04.2

備考
そろそろサーバーのスペック不足が著しくなってきたなー、Minecraftサーバーも快適に動かない。 せや！強化したろ、って思ったんで友人に頼み込んで用意してもらいました。 スペックはおそらく所持鯖の中では最も高いと思ってます。実際にメモリはSSD並ですし... CPUも豪華に2つ載せ、ストレージも十分に余裕をもたせてます。 今はまだMinecraft鯖を動かしてないんですが、前にテストで立てた鯖は昔以上に快適になっていました。 非常に満足してます:D

備考
小型サーバーとして2019年から運用開始、電気代を大幅に抑えて運用することが可能な上、 その後いくつかの改造手当や延命措置、OSでの動作等を変更し、現在の構成に至る。 最近ではマイクラサーバー等の高性能を要求する動作をしなくなったため、現状構成で 落ち着いている。おそらくあと１年弱はこのままの構成と思われる。 将来的には２つ動いているサーバーをまとめるために高性能小型PCを導入し、 スペック向上と容量不足の問題を解決する予定です。

“HP Server“ (コードネーム:仮バックアップ鯖機)
稼働期間: 2020のどっか~2020/10/14
CPU: Intel Core2Duo E8400 2C2T@3.00GHz
GPU: オンボード
RAM: DDR3 2GB x2
Storage: どっかのHDD 250GB
OEM: HP Compaq 8000 Elite USDT (LGA775)
OS: Ubuntu Linux 20.04

備考
もともとはサブPCとして使う予定だったPCです。 導入後にこのPC以上に高性能なものが導入されたため、ご退役しました。 その後は何度か用途を変更(あるときはマイクラ鯖、あるときは家族専用PC)したのちに サーバーの容量不足を解決すべく、分散用としてバックアップサーバーとして再導入されました。 そのうち資金が溜まったら完全引退して、コレクションされる予定です。 ちなみにこの鯖を買った当初、電源コネクタは付いてませんでした。
